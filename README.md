# Data Analytics Portfolio

## Week 1 — World Happiness Report Analysis

### Tools Used
- Python, Pandas, NumPy, Matplotlib, Seaborn

### Dataset
- World Happiness Report 2024 (Kaggle)

### Key Findings
1. GDP influences happiness but doesn't determine it — Costa Rica ranks top 5 despite modest wealth
2. Social support is the strongest driver of happiness across all factors analysed
3. Generosity has almost no correlation with happiness (0.06)

### How to Run
1. Clone this repository
2. Install requirements: `pip install pandas numpy matplotlib seaborn`
3. Open `week1_analysis.ipynb` in Jupyter Notebook
